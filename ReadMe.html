To install pymunk and pygame on Windows:

"C:\Program Files (x86)\Python35-32\python.exe" -m pip install pygame
"C:\Program Files (x86)\Python35-32\python.exe" -m pip install pymunk

(You need to be an administrator.)

To install pymunk and pygame on Linux:

pip3 install pygame
pip3 install pymunk

(You don't need to be an administrator.)

To actually run the simulation:

python3 Controller.py --test Robot1Code.py Robot2Code.py ...
